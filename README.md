# 贪吃蛇小游戏

一个使用Python标准库tkinter实现的经典贪吃蛇游戏。

## 运行方式

确保安装了Python 3.6或更高版本，然后运行：

```bash
python game.py
```

或

```bash
python3 game.py
```

## 游戏玩法

### 操作控制

- **方向键 ↑/↓/←/→**: 控制蛇的移动方向
- **空格键**: 游戏结束后重新开始

### 游戏规则

1. 使用方向键控制绿色蛇头的移动方向
2. 吃到红色食物可以增加蛇的长度和分数
3. 撞到墙壁或自己的身体则游戏结束
4. 目标是获得尽可能高的分数

### 计分规则

- 每吃一个食物得1分

## 技术特点

- 纯Python标准库实现，无需安装任何第三方依赖
- 使用tkinter创建图形界面
- 流畅的动画效果
- 实时计分显示

## 自检

可以通过以下命令检查代码语法：

```bash
python -m py_compile game.py
```
